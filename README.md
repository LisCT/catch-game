# Canvas Boilerplate


## Repository

1. Install dependencies

```bash
yarn install or npm install
```

2. Run Build Development (Compile public folder)

```bash
yarn run webpack or npm run webpack
```

Running local config in webpack.config.js

```bash
http://localhost:3000
```
